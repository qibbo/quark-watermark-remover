# 夸克去水印 Android 版设计文档

## 概述

去除夸克扫描王 PDF 中的水印，纯本地处理，无需联网。

## 技术选型

| 项目 | 选择 |
|------|------|
| 语言 | Kotlin |
| 最低版本 | Android 8.0 (API 26) |
| 构建工具 | Gradle + Android Studio |
| PDF 库 | iText 7 (AGPL，非商用) |
| UI 框架 | Material3 + Jetpack Compose |
| 主题 | 浅色主题 |

## 功能需求

### 输入方式

1. **手动选择**：点击按钮，调用 Android 文件选择器（SAF），选择 PDF 文件
2. **接收分享**：注册 Intent Filter（`ACTION_SEND`，`application/pdf`），出现在其他应用的分享列表中

### 输出方式

1. **保存到 Downloads**：默认保存到 `Downloads/夸克去水印/` 目录，文件名格式 `{原名}_去水印.pdf`
2. **分享给其他应用**：处理完成后可选择分享

### 核心处理逻辑

使用 iText 7 读取 PDF 内容流，正则匹配并移除水印命令：

- 水印模式：`q\s+[\d\s]+cm\s+/QuarkX2\s+Do\s+Q`
- 遍历每页的 `/Contents` 流
- 移除匹配内容，清理多余空行
- 保存新 PDF

### 结果汇总

处理完成后弹窗显示：

- 成功数量
- 失败数量
- 失败详情（文件名 + 原因）
- 典型错误：文件已加密、文件损坏、无水印
