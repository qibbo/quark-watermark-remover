# 夸克扫描王 PDF 去水印工具 - 设计文档

## 概述

批量去除夸克扫描王非会员版生成的 PDF 文件中的水印，打包为独立 EXE，同事可直接使用。

## 技术选型

### 选定方案：PyMuPDF + CustomTkinter

- **PDF 处理**：PyMuPDF（fitz）- 高效读写 PDF
- **GUI 框架**：CustomTkinter - 现代美观的 tkinter 扩展
- **打包工具**：PyInstaller - 生成独立 EXE
- **预计体积**：15-20 MB

### 备选方案：PyMuPDF + tkinter 原生（方案 C）

如后期需要更轻量的版本，可切换为 tkinter 原生：

- **优点**：无额外 UI 依赖，打包最小（约 10-15 MB）
- **缺点**：UI 较简陋，拖拽支持需额外处理
- **适用场景**：追求极致轻量，或 CustomTkinter 出现兼容问题时

### 交互设计

- **添加文件**：拖拽 PDF 到窗口，或点击选择（支持多选）
- **输出路径**：默认为原文件目录，可手动设置，点击「恢复默认」改回原目录
- **开始处理**：点击「开始去水印」，进度条显示进度，列表实时更新状态
- **清空列表**：点击「清空列表」移除所有文件

## 水印删除逻辑

### 水印特征

经分析测试文件，夸克扫描王水印特征如下：

- **类型**：图片水印（PNG 格式，351×108 像素）
- **位置**：每页右下角（坐标约 389,849 至 551,899）
- **引用方式**：页面内容流中的 `/QuarkX2 Do` 命令
- **共享机制**：所有页面共享同一个水印图片对象（xref=5）

### 删除流程

1. 打开 PDF 文件
2. 遍历每一页：
   - 获取页面内容流
   - 查找包含 `QuarkX2` 的行
   - 删除该行（水印绘制命令）
3. 保存为新文件

### 水印检测策略

- **主要检测**：查找内容流中的 `QuarkX2` 图片引用
- **备用检测**：查找右下角区域的小尺寸图片（宽<500 且 高<200）

## 文件命名与冲突处理

### 输出文件命名

- 格式：`原文件名_去水印.pdf`
- 示例：`单店合同.pdf` → `单店合同_去水印.pdf`

### 文件冲突处理

当输出目录已存在同名文件时：

- 自动重命名：`原文件名_去水印(2).pdf`
- 依次递增：`原文件名_去水印(3).pdf` ...

## 错误处理

### 错误隔离机制

- 每个文件独立处理，互不影响
- 单个文件失败 → 标记为「失败」+ 显示错误原因，继续处理下一个
- 所有文件处理完成后 → 显示汇总（成功 X 个，失败 Y 个）

### 常见错误场景

| 场景 | 处理方式 |
|------|----------|
| 文件被其他程序占用 | 提示「文件被其他程序占用」，继续处理其他文件 |
| 文件损坏或无法读取 | 提示「文件损坏」，继续处理其他文件 |
| 文件无水印 | 标记为「无需处理」，继续处理其他文件 |
| 输出路径无写入权限 | 提示「无写入权限」，继续处理其他文件 |

## 配置持久化

### 保存的配置项

- 输出路径（上次设置的路径）
- 窗口大小和位置

### 存储方式

- 文件格式：JSON
- 存储位置：程序同目录下的 `config.json`
- 首次运行自动创建，之后读取已有配置

### 恢复默认

- 点击「恢复默认」按钮
- 输出路径清空（使用原文件目录）
- 同时清除保存的路径配置

## 打包发布

### PyInstaller 打包

- 打包为单个 EXE 文件
- 包含所有依赖（CustomTkinter、PyMuPDF 等）
- 同事无需安装 Python 或任何依赖

### 打包命令

```bash
pyinstaller --onefile --windowed --name "夸克去水印" main.py
```
