# 企业微信 PDF 自动处理助手（简版方案）

## 需求

我和同事两个人使用。

流程：

1. 在企业微信中发送 PDF 文件。
2. 云服务器自动接收文件。
3. 调用现有 Python（pypdf）脚本处理 PDF。
4. 将处理后的 PDF 自动发回发送者。

使用频率不高，不要求高并发。

---

## 技术方案

### 服务器

使用 Oracle Cloud Free Tier 免费服务器。

原因：

* 长期免费。
* 性能足够。
* 适合运行 Python 服务。

---

### 企业微信

使用「自建应用」。

原因：

* 支持接收文件。
* 支持自动发送文件。
* 更适合文件处理流程。

不使用群机器人。

---

### 服务组成

```text
企业微信
    ↓
Oracle服务器
    ↓
Python + pypdf
    ↓
企业微信
```

---

## 服务器功能

* 接收 PDF
* 调用现有处理脚本
* 返回处理结果
* 记录简单日志

不使用：

* 数据库
* Redis
* 消息队列

---

## 域名
